Bem-vindo ao Pratica Cargo

Sistema desenvolvido para praticar Angular 14 de forma organizada, didática e totalmente com fins de Aprendizado!

🚀 Sobre o Projeto

O Pratica Cargo é um sistema fictício criado com o objetivo de praticar conceitos fundamentais do Angular, simulando um sistema real de gestão logística.

📦 O que você pode fazer

  • Gerenciar motoristas
  • Gerenciar empresas
  • Criar ordens de serviço (carga e descarga)
  • Consultar dados de forma organizada

🛠 Tecnologias utilizadas

  • Angular 14
  • Angular Material
  • Reactive Forms
  • json-server (API fake)

📚 Objetivo de Aprendizado

Este projeto foi desenvolvido para consolidar conhecimentos em Angular, boas práticas de organização de código, componentização, serviços e rotas.
